Instalación de Active Directory en Windows Server 2019

Configuración previa

Todo se puede realizar a través del apartado "Servidor local":

  1. Cambiar nombre del equipo al que queramos.
  2. Establecemos una IP fija.
  3. Revisamos que la zona horaria sea la correcta.

Implementación de Active Directory

  1. En el panel principal, le damos a "Agregar roles y características. Se desplegará un menú y continuamos con la instalación.
  2. En el apartado de roles, seleccionamos "Servidor DNS" y "Servicios de dominio de Active Directory". Continuamos e instalamos.
  3. Una vez instalado, en la parte superior nos saldrá un aviso. Lo abrimos y le damos a "Promover este servidor a controlador de dominio"
  4. Agregamos un nuevo bosque y ponemos el dominio raíz que deseamos (formato xxxx.xxx). Le damos a siguiente.
  5. Establecemos una contraseña que cumpla con los requisitos. Le damos a siguiente hasta instalar.
  6. El servidor se reiniciará automáticamente y ya estaría implementado.

Creación de Unidades Organizativas, grupos y usuarios

  1. Arriba a la derecha, en "Herramientas", seleccionamos la opción de "Usuarios y equipos de Active Directory".
  2. Seleccionamos el servidor.
  3. Para crear una Unidad Organizativa, damos Click derecho sobre el servidor, "Nuevo" y "Unidad organizativa". La creamos con el nombre correspondiente
  4. Si queremos crear un usuario, damos click derecho sobre la UO y nuevo usuario. Cumplimentamos toda la información que nos pide.
  5. Para crear un grupo, seguimos la misma dinámica, pero creando el grupo.
  6. Si queremos añadir un usuario a un grupo, damos click derecho sobre el usuario y seleccionamos "Agregar a un grupo". Seleccionamos los grupos a los que queremos que pertenezca y aceptamos.
