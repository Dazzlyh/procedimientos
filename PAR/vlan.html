Pasos para la creación de una VLAN

  1. Primeramente, tenemos que establecer:
    • Las diferentes VLAN
    • Los dispositivos que tendremos
    • El direccionamiento IP
    • La topología de red
  2. Crear VLANs en el Switch:
    • enable
    • configure terminal
    • Creamos VLAN: vlan [número]
    • Ponemos nombre: name [NOMBRE]
    • exit
    • Repetimos con todas las VLANs
  3. Configurar modo acceso:
    • enable
    • configure terminal
    • Entramos en la interfaz que queremos configurar: interface range fastEthernet 0/X-X
    • switchport mode access
    • switchport access vlan [numero_VLAN]
    • Repetimos este paso con todas las VLAN en los diferentes puertos
  4. Configurar modo trunk:
    • En cada uno de los puertos del switch por el que vayan a pasar varias VLAN:
    • enable
    • configure terminal
    • interface fastEthernet 0/X
    • switchport mode trunk
    • (Opcional según dispositivo) switchport trunk encapsulation dot1q
  5. Configurar Router (Router-on-a-Stick):
    • enable
    • configure terminal
    • Entramos en la interfaz que conecta con el switch: interface fastEthernet 0/X
    • no shutdown
    • exit
  6. Configurar Subinterfaces del Router:
    • Entramos en la subinterfaz: interface fastEthernet 0/X.[número_VLAN]
    • encapsulation dot1Q [número_VLAN]
    • ip address [direccion_IP] [máscara]
    • Repetimos proceso con todas las VLANs
  7. Asignar IPs a los diferentes dispositivos:
    • Asignar IP dentro de su VLAN
    • Asignar máscara de red
    • Asignar Gateway (IP de la subinterfaz del router)
  8. Verificación de la configuración:
    • En Switch: show vlan brief
    • En Switch: show interfaces trunk
    • En Router: show ip interface brief
    • Realizar ping entre equipos de la misma VLAN
    • Realizar ping entre VLANs
  9. Guardar configuración:
    • En Switch: copy running-config startup-config
    • En Router: copy running-config startup-config
