Comandos para administración de la red

netstat

Muestra un listado en tiempo real de todas las conexiones activas de tu PC (tanto las que entran como las que salen)

netstat [-parámetros] - muestra el listado utilizando los filtros indicados:

  • -a muestra todas las redes (activas/inactivas) para identificar problemas
  • -e estadísticas de paquetes entrantes y desviados
  • -n muestra números de puerto en lugar de nombres
  • -o muestra el PID de cada conexión
  • -p [protocolo] filtra por el protocolo indicado (UDP, TCP...)
  • -r muestra la tabla de enrutamiento
  • -s estadísticas por protocolo (IPv4, IPv6, ICMP...)
  • -t datos de conexiones en estado de descarga (offload)

hpin3 (Linux) / nping (Windows)

Es una herramienta avanzada (evolución del ping) que permite modificar paquetes TCP/IP.

  • Modos:
    • -0 RawIP
    • -1 ICMP
    • -2 UDP
    • -8 Escanéo de puertos
  • Flags TCP:
    • -S SYN
    • -A ACK
    • -F FIN

Saltarse el Firewall: lo que se hace es "Enviar 4 paquetes (-c 4) dirigidos al puerto 80 (-p 80) y únicamente la bandera SYN (-S o --flags syn)"

  • sudo hping3 -S -p 80 -c 4 [IP_Destino]
  • nping --tcp -p 80 --flags syn -c 4 [IP_Destino]

Ataque SYN Flood (Denegación de Servicio):

  • sudo hping3 -S -p 80 --flood --rang-source [IP_Destino] -- flood dispara a la máxima velocidad posible sin esperar respuesta. --rang-source falsifica la IP de origen en cada paquete
  • nping --tcp -p 80 --flags syn --rate 5000 [IP_Destino] --rate indica el número de paquetes que se van a enviar

nmap

Es un software libre para el descubrimiento de redes y auditoría de seguridad: descubrir equipos (hosts), identificar puertos abiertos y servicios, detectar el SO y versión y configurar uso de Firewall.

nmap [IP_destino] realiza el escaneo
