Guía de Etiquetas HTML (Temas 5 y 6)

1. Estructura Básica y Metadatos

  • <!DOCTYPE html> - Indica al navegador que el documento utiliza la versión HTML5 [1, 2].
  • <html> - Define el inicio del documento HTML y abarca todo su contenido [1].
  • <head> - Contiene información técnica y metadatos no visibles del documento [1].
  • <body> - Contiene todo el contenido visible de la página web para el usuario [3].
  • <title> - Establece el título que se muestra en la pestaña del navegador [3].
  • <meta> - Define propiedades como la codificación de caracteres o la vista en dispositivos [3, 4].
  • <link> - Se utiliza para enlazar recursos externos, como hojas de estilo CSS [3].

2. Etiquetas de Texto y Formato

  • <h1> a <h6> - Definen diferentes niveles de encabezado o títulos jerárquicos [5].
  • <p> - Define un párrafo de texto [5].
  • <b> - Aplica un formato de negrita visual al texto [5].
  • <i> - Aplica un formato de cursiva visual al texto [5].
  • <u> - Aplica un subrayado al texto [5].
  • <strong> - Define texto importante con un fuerte énfasis semántico [5].
  • <em> - Destaca el texto con un énfasis semántico suave (cursiva) [5].
  • <small> - Disminuye el tamaño del texto respecto al estándar [5].
  • <sub> - Da formato de subíndice al texto [5].
  • <sup> - Da formato de superíndice al texto [5].
  • <blockquote> - Se utiliza para realizar sangrados en bloques de texto o citas largas [5].
  • <br> - Crea un salto de línea manual dentro del contenido [5].
  • <pre> - Permite que el texto se visualice respetando espacios y saltos del código fuente [5].
  • <code> - Indica que el contenido es un fragmento de código de programación [5].

3. Listas, Enlaces e Imágenes

  • <a> - Define un hipervínculo para redirigir a otras páginas o secciones internas [5].
  • <img> - Define una imagen para ser mostrada en el documento [5].
  • <ul> - Inicia una lista desordenada que utiliza viñetas [5].
  • <ol> - Inicia una lista ordenada que utiliza numeración o letras [5].
  • <li> - Indica un elemento individual dentro de cualquier tipo de lista [5].

4. Formularios y Validación

  • <form> - Define el contenedor principal para la interacción y envío de datos [6].
  • <fieldset> - Agrupa elementos relacionados dentro de un recuadro organizativo [6].
  • <legend> - Proporciona un título descriptivo al grupo de elementos en un fieldset [7].
  • <label> - Etiqueta descriptiva que ayuda a identificar campos de entrada específicos [7].
  • <input> - Define campos de entrada de datos versátiles (texto, número, radio, etc.) [7, 8].
  • <select> - Crea una lista desplegable para elegir una opción [9].
  • <option> - Define cada una de las opciones disponibles dentro de un select [9].
  • <textarea> - Permite el ingreso de texto libre en varias líneas [9].

5. Etiquetas Semánticas Modernas (HTML5)

  • <header> - Define la cabecera de la página o de una sección específica [10].
  • <nav> - Bloque que contiene los enlaces principales de navegación del sitio [10].
  • <main> - Define el contenido principal y único del cuerpo del documento [11].
  • <section> - Agrupa contenido temáticamente relacionado dentro de la página [12].
  • <article> - Define contenido independiente que tiene sentido por sí mismo (noticias, blogs) [12].
  • <aside> - Representa contenido complementario o secundario, como barras laterales [13].
  • <footer> - Define el pie de página o la parte inferior de una sección [13].

6. Multimedia y Contenido Avanzado

  • <audio> - Permite la reproducción nativa de archivos de sonido [14].
  • <video> - Permite la inclusión y reproducción nativa de contenido de vídeo [15].
  • <source> - Define los archivos multimedia y sus formatos para asegurar compatibilidad [14, 15].
  • <track> - Permite añadir subtítulos, transcripciones o capítulos a un vídeo [16].
  • <!-- --> - Se utiliza para incluir comentarios que no son visibles en el navegador [17].
