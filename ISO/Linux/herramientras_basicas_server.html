Herramientas básicas para administrar servidores Linux

cURL

cURL es una herramienta de línea de comandos que permite transferir datos desde o hacia un servidor usando protocolos como HTTP, HTTPS, FTP o SSH. Se utiliza comúnmente para probar APIs, descargar archivos o comprobar servicios web.

htop

Es un monitor interactivo del sistema que permite ver procesos, uso de CPU, memoria y carga del sistema en tiempo real, con una interfaz más clara y manejable que el comando top

net-tools

Es un conjunto de herramientas de red clásicas en Linux que incluye comandos como ifconfig, netstat, route y arp, utilizados para configurar y diagnosticar redes.

git

Es un sistema de control de versiones distribuido que permite gestionar cambios en archivos y colaborar en proyectos, manteniendo un historial de modificaciones y facilitando el trabajo en equipo.
