Gestión de usuarios

Listar usuarios

cat /etc/passwd - visualizas todos los usuarios creados (a partir del 1000)

Creación de usuarios

sudo adduser [nombre] - crea el usuario

sudo passwd [usuario] - modifica la contraseña del usuario

Administración de usuarios

sudo userdel [usuario] - elimina el usuario indicado

Grupos

sudo groupadd [nombre_grupo] - crea el grupo con el nombre indicado

sudo usermod -aG [nombre_grupo] [usuario] - añade al usuario al grupo indicado

groups [usuario] - indica los grupos a los que pertenece el usuario indicado

gpasswd -d [usuario] [nombre_grupo] - elimina al usuario del grupo indicado
