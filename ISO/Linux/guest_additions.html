Instalar Guest Additions en UBUNTU/DEBIAN

  1. Actualizar listado de paquetes: sudo apt update
  2. sudo apt install build-essential dkms linux-headers-$(uname -r)
  3. Insertar disco Guest Additions: Dispositivos -> Insertar imagen de CD de las Guest Additions
  4. ls /media
  5. cd /media/&USER/VBox_GAs*
  6. sudo sh VBoxLinuxAdditions.run
  7. sudo reboot
