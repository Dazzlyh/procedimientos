Instalación de OpenSSH

  1. Revisamos si está instalado. Para ello, abrimos PowerShell y ponemos el comando Get-WindowsCapability -Online -Name Open*. Esto revisará todo lo que tenga Open como nombre, y si está instalado. Lo más normal es que no esté instalado el OpenSSH.Server
  2. Para instalarlo, usamos el comando Add-WindowsCapability -Online -Name OpenSSH.Server~~~~0.0.1.0. Podemos comprobar nuevamente si se habrá instalado.
  3. A continuación, debemos inicializar los servicios ya que estarán parados. Podemos comprobar el estado con Get-Service sshd y Get-Service ssh-agent, que deberían aparecer como Stopped.
  4. Para configurarlos, debemos usar los siguientes comandos:
    Set-Service -Name sshd -StartupType Automatic
    Set-Service -Name ssh-agent -StartupType Automatic
  5. Para inicializar los servicios, usamos:
    Start-Service sshd
    Start-Service ssh-agent
  6. Ya estaría instalado listo para permitir el acceso remoto.
