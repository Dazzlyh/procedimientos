Instalar servidor DHCP

Un servidor DHCP se encarga de la asignación y administración automática de direccines IP y configuraciones de red relacionadas.

Pasos para la instalación

  1. Agregamos el rol de Servidor DHCP en las opciones. Tras la instalación, aparecerá en la parte inferior.
  2. Nos habrá saltado un error en la parte superior. Seleccionamos para completar la instalación. Siguiente y cerramos.
  3. Para configurarlo, vamos a Herramientras -> DHCP. Se abrirá una ventana.
  4. Desplegamos el servidor y, sobre IPv4 damos click derecho -> Ámbito nuevo
  5. Tenemos que poner un nombre, el que queramos. Damos a siguiente.
  6. Tendremos que poner el rango que IPs que queremos que se asignen automáticamente
  7. Si queremos, podemos excluir alguna IP en la siguiente opción.
  8. También tenemos que indicar cada cuántos días queremos que se renueven las IPs
  9. Le damos a siguiente hasta que tengamos que poner la gateway.
  10. Tenemos que revisar que el dominio está bien configurado y que resuelve correctamente. Vamos hasta finalizar.
  11. Reiniciamos el servidor, para evitar problemas.
  12. Tras esto, ya debería de estar asignando las IPs de manera automática en los equipos que estén en el dominio.
